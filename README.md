# 🎬 Disney+ Content Analysis Dashboard

## 📖 Overview

This project presents an interactive data visualization dashboard built using Power BI to analyze Disney+ content. The objective is to explore patterns in movies and series by examining ratings, release trends, genres, and audience engagement.

The dashboard enables users to quickly understand how content is distributed across different categories and identify high-performing titles.

---

## 🎯 Objectives

* Analyze the distribution of movies and series
* Identify top-rated and popular content
* Understand trends in content release over time
* Explore audience engagement using IMDb votes
* Provide an interactive experience using filters

---

## 🛠️ Technologies Used

* **Microsoft Excel** → Data cleaning and preparation
* **Power BI** → Data visualization and dashboard creation

---

## 📂 Dataset Information

The dataset consists of Disney+ titles with multiple attributes that help in analyzing content performance.

### 🔑 Key Fields:

* **Title** – Name of the show/movie
* **Type** – Movie or Series
* **Genre** – Category of content
* **IMDb Rating** – Viewer rating score
* **IMDb Votes** – Number of user votes
* **Year** – Release year
* **Language** – Language of the content
* **Country** – Production country
* **Runtime** – Duration of content

---

## 🔄 Data Preparation

* Removed missing and duplicate values
* Ensured proper formatting of columns
* Converted data types (numeric, text, year)
* Structured dataset for easy visualization in Power BI

---

## 📊 Dashboard Components

### 🔹 KPI Indicators

* Total number of titles
* Total movies and series
* Average IMDb rating

### 🔹 Visualizations

* **Bar Chart** → Displays top-rated titles
* **Donut Chart** → Shows distribution of movies vs series
* **Line Chart** → Tracks number of releases over the years
* **Map Visualization** → Highlights content distribution by country

### 🔹 Filters (Slicers)

* Genre
* Year
* Country

These filters allow users to dynamically explore the dataset.

---

## 📈 Key Insights

* Movies represent a significant portion of the platform's content
* A small number of titles dominate in terms of ratings and votes
* Content production has increased over recent years
* Audience engagement varies widely across different titles
* Certain countries contribute more to content production

---

## 🎨 Dashboard Design Approach

The dashboard is designed with a clean and structured layout:

* Top section highlights key metrics
* Middle section focuses on comparison visuals
* Bottom section provides trend and geographic insights
* Right panel includes interactive filters

This layout ensures clarity and ease of navigation for users.
