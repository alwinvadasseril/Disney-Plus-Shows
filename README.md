# Disney-Plus-Shows
📌 Project Overview

This project analyzes Disney+ content using Excel and Power BI to understand content distribution, ratings, and audience engagement. The dashboard provides an interactive view of movies and series based on genre, release year, ratings, and popularity.

🛠️ Tools Used
Microsoft Excel
Power BI
📊 Dataset
Source: Sample Disney+ dataset
Key Columns:
Title
Type (Movie / Series)
Genre
IMDb Rating
IMDb Votes
Year
Language
Country
Runtime
🔄 Steps Followed
Cleaned data in Excel (removed blanks, formatted columns)
Verified correct data types (Year, Rating, Votes, etc.)
Imported dataset into Power BI
Built dashboard using:
KPI Cards
Bar Chart (Top Rated Titles)
Donut Chart (Movies vs Series)
Line Chart (Titles by Year)
Map (Titles by Country)
Added slicers for:
Genre
Year
Country
Applied Top N filtering and sorting
Designed a clean and structured dashboard layout
📈 Key Insights
Movies form the majority of content compared to series
Top-rated titles stand out with significantly higher IMDb ratings
Content production has increased in recent years
Audience engagement is concentrated on a few highly popular titles
Most content is produced in a limited number of countries
